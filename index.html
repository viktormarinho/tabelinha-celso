Tabela de comandos principais Windows DOS

Trabalho de SOP - Viktor Marinho e Gustavo Baltazar

Comando Descrição
dir Lista itens do diretório atual
cd Troca de diretório
mkdir Cria um ou mais diretórios
ipconfig Exibe todas suas configurações de IP
exit Fecha o prompt de comando
del Deleta itens do diretório atual
date Exibe a data atual e permite que você a mude
time Exibe a hora atual e permite que você a mude
ver Exibe a versão do sistema operacional
cls Limpa a tela do prompt de comando
rd Remove pasta caso ela esteja vazia
chkdsk Verificador de erros do HD
mem Exibe informações da memória RAM
rename Renomeia arquivo
copy Cpopia um ou mais arquivos para algum diretório
move Move arquivos para algum diretório
format Apaga todo o conteúdo de um disco
tree Exibe graficamente os diretórios e subdiretórios
color Permite a mudança das cores do terminal
